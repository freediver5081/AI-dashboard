💡 핵심 개념

CLAUDE.md란?

프로젝트 폴더에 넣어두는 "프로젝트에 대한 설명서"

🔥 함께 해보기

CLAUDE.md 만들기

1 자연어로 요청

> 글로벌 AI 서비스(OpenAI, Anthropic, Google, Microsoft) 최신 뉴스를 한국어로, 최신순으로 정리하는 대시보드를 만들거야. CLAUDE.md 만들어줘

2 /init 한 줄

> /init

→ 폴더 안 파일을 스캔해서 프로젝트 구조 기반으로 CLAUDE.md 자동 작성
(이미 만든 파일이 있을 때 추천)

💡 만들어진 CLAUDE.md를 열어서 한 줄만 직접 추가해보세요
예: "응답은 항상 한국어로", "이모지 사용 금지" 등
→ 다음 대화부터 Claude가 그 규칙을 자동으로 따릅니다

CLAUDE.md 예시

# CLAUDE.md — 글로벌 AI 서비스 트렌드 대시보드 ## 프로젝트 개요 OpenAI · Anthropic · Google · Microsoft 등 주요 AI 서비스의 최신 업데이트·뉴스를 수집해 대시보드로 보여주는 프로젝트. ## 출력 규칙 - 모든 응답과 코드 주석은 한국어로 - 뉴스는 항상 최신순으로 정렬 - 서비스별로 카드를 명확히 구분 ## 금지 사항 - 출처 날짜가 검증되지 않은 뉴스 수록 금지 - AI 기업 주가·투자 분석 같은 일반 뉴스 포함 금지

💡 핵심은 "무엇을 만들지"와 "어떻게 만들지" 두 가지

CLAUDE.md 추가 전 vs 후

❌ Before (맥락 없음)

"어떤 AI 서비스를 분석할까요?"
영어로 응답
관련 없는 뉴스 포함
정렬 기준 불명확

✅ After (CLAUDE.md)

OpenAI·Anthropic·Google 등 자동 선택
전부 한국어
최신순 정렬
바로 작업 시작

뉴스 검색 → 정리 → HTML 카드 UI → GitHub + Vercel 배포 → 내 URL

뉴스 수집 후 대시보드 만들기

> OpenAI·Anthropic·Google·Microsoft 최신 AI 뉴스 검색해서 index.html 만들어줘. AI 서비스별 카드 형태로 보여주고 최신순 정렬해줘. UI는 shadcn/ui (ui.shadcn.com) 스타일로 깔끔하게 만들어줘

🌐 이때 일어나는 일

Claude Code 내장 WebSearch가 실시간으로 뉴스를 검색합니다.
별도 MCP 설치 없이 바로 동작!

🎨 UI 라이브러리 — 왜 shadcn/ui?

ui.shadcn.com — 사실상 업계 표준. 문서·예제·커뮤니티가 풍부해 Claude Code가 가장 잘 이해합니다.
참고: ui.aceternity.com은 화려한 애니메이션이 강점이지만 설정이 복잡해 입문에는 shadcn/ui가 유리합니다.
